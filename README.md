### Pre-requisites ðŸ”´
- [Docker](https://docs.docker.com/docker-for-windows/install/)
- [Yarn](https://yarnpkg.com/)
- [Node](https://nodejs.org/en/download/)

### Installation 
1- to run the client app
```bash
  cd client 
  yarn start
```

2- to run the server
```bash
  cd api
  docker build -t sandbox .
  yarn dev
```
